Big users of graph analysis to (literally) tackle real world "problems"

Data Professionals

It might seem strange, but learning how to draw circles and arrows effectively is a very useful transferable skill.

When describing a complex system, many people resort to diagrams with circles and arrows. Circles and arrows leave one free to describe the interrelationships between things in a way that tables, for example, do not. The system we need is like a diagram of circles and arrows, where circles and arrows can stand for anything.

Tim Berners-Lee https://www.w3.org/History/1989/proposal.html

Circles and arrows were the inspiration for the WWW.

Let's learn how these circles and arrows work, then connect them to other networks we have discussed.

This is especially relevant for essay questions 1 and 3 on the internet, WWW, HTML and HTTP.
